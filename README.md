## Requirements

* [Node.js](v6 or higher)
* [HapiJS] 17
* [Mongoose]
* [handlebars]


## Installation Steps

1. Clone repo
2. Run `npm install`
3. Run `npm run start:dev`
4. Visit http://localhost:3000/
